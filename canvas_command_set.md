# SYSTEM ET CANVAS COMMAND SET


## ZOOMING
Zoom to fit:
```
zoom fit
```

Zoom by:
```
zoom level 3
```


Zoom on point:
```
zoom center 10 10 # x/y
```

```
zoom center 10 10 2 # x/y scale
```


## SHOW
Show device in schematic requires device name and unit name:
```
show device R1 1
```

```
show device IC1 IO-BANK2
```

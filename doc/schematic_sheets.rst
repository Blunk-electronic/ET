Sheets
------

Show a sheet
^^^^^^^^^^^^

	.. code-block::

		show sheet 2

	|VNS| |NI|


	
Set Sheet Category
^^^^^^^^^^^^^^^^^^

	.. code-block::

		set sheet 2 development


	.. code-block::

		set sheet 2 routing


	.. code-block::

		set sheet 2 product





Delete a Sheet
^^^^^^^^^^^^^^

	This example command deletes a sheet incl. everything
	on it:

	.. code-block::

		delete sheet 2

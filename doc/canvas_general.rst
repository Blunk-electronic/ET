********************************
General Navigating on the Canvas
********************************


++++
Grid
++++

The commandline allows setting the properties of the grid.

#. Enabling and Disabling

	.. code-block::

		set grid on
		set grid off


#. Spacing

	.. code-block::

		set grid spacing 5
		set grid spacing 5 10


#. Style

	.. code-block::

		set grid style dots
		set grid style lines


	


	
++++++
Cursor
++++++


#. Placing

	.. code-block::

		zoom cursor 10 10


#. Placing and Zooming

	.. code-block::

		zoom cursor 10 10 50


		
	
++++
Zoom
++++


#. Setting Zoom Factor

	.. code-block::

		set zoom 50


	
#. Zoom-to-Fit / Fit All

	.. code-block::

		zoom all

		
		
+++++
Scale
+++++

#. Setting the Scale

	.. code-block::

		set scale 10

	After setting the scale, a default grid spacing
	is applied automatically. The user is required to
	set the grid spacing anew.

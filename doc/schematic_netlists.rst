Netlists
--------

Export netlist
^^^^^^^^^^^^^^


	This example command exports a category 1 netlist
	of the module in the default assembly variant:


	.. code-block::

		make netlist default 1



	This example command exports a category 1 netlist
	of the module in the given assembly variant "low_cost":


	.. code-block::

		make netlist low_cost 1

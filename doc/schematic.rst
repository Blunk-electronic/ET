####################################
Commands and Verb-Noun key sequences
####################################

+++++++++++++++++
Devices and Units
+++++++++++++++++

#. Add device

	.. code-block::

		add device

	Verb-Noun key sequence: a d

	
#. Delete unit

	.. code-block::

		delete unit IC1 [B]

	Verb-Noun key sequence: del u
	

#. Drag unit

	.. code-block::

		drag unit IC1 [A]

	Verb-Noun key sequence: d u


#. Invoke unit

	.. code-block::

		invoke unit IC1 [B]

	Verb-Noun key sequence: i u

	
#. Move unit

	.. code-block::

		move unit IC1 [A]

	Verb-Noun key sequence: m u
	

#. Rotate unit

	.. code-block::

		rotate unit IC1 [A]

	Verb-Noun key sequence: r u


#. Set package variant

	.. code-block::

		set variant IC1

	Verb-Noun key sequence: s a

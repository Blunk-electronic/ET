####################################
Commands and Verb-Noun key sequences
####################################

+++++++++++++++++
Devices and Units
+++++++++++++++++

#. Add device

	.. code-block::

		add device

	Verb-Noun key sequence: a d

	
	
#. Invoke unit

	.. code-block::

		invoke unit IC1 [B]

	Verb-Noun key sequence: i u

	
	
#. Delete unit

	.. code-block::

		delete unit IC1 B

	Verb-Noun key sequence: del u


	
#. Move unit

	.. code-block::

		move unit IC1 [A]

	Verb-Noun key sequence: m u
	
	
	

#. Drag unit

	.. code-block::

		drag unit IC1 [A]

	Verb-Noun key sequence: d u

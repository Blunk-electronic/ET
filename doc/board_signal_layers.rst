Signal Layers
-------------

Adding a Signal Layer
^^^^^^^^^^^^^^^^^^^^^

	This example adds a signal layer with
	conductor thickness of 0.035mm and a dielectric
	thickness of 0.2mm:
	
	.. code-block::
	
		add layer 0.035 0.2
		

Deleting a Signal Layer
^^^^^^^^^^^^^^^^^^^^^^^

	This example deletes signal layer 2:
	
	.. code-block::
	
		delete layer 2
